# ptm

#### Description
ptm(Packaging tools mirrors)即全自动配置包管理工具的镜像网站，灵感来源于pacman-mirrors
ptm提供镜像站速度、测试，自动挑选并设置最优镜像站
目前支持:
yum、apt、zypper
项目目前仍在开发中
